# Technologies and Instruction

The course covers a full software stack of technologies and topics.

- Command console
- Java
- HTTP
- Web Services
- Data Services demonstrated with MySQL
- Realtime peer to peer interaction with WebSocket
- Security
- Testing
- Application design
